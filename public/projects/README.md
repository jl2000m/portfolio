# Capturas de proyectos

Añade imágenes aquí y enlázalas en `lib/projects.ts` como `coverImage`.

**Convención sugerida**

- Nombre: `{slug}.webp` o `{slug}.png` (ej. `birriapp.webp`)
- En datos: `coverImage: "/projects/birriapp.webp"`
- Ratio recomendado: **16:10** o **3:2**, ancho ≥ 1200px para que se vea nítido.

**Vista embebida (opcional)**

Si el sitio permite `iframe` (p. ej. algunos prototipos en Figma, Loom), puedes usar `previewEmbedUrl`. Muchas URLs normales bloquean el embed; en ese caso deja solo `coverImage` o `liveUrl`.
